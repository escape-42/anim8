📷 Upload ✎ Draw

📚 Help

📷 Upload Mode

  • Drag & drop or click to select 2–8 images
  • Supported formats: PNG, JPG, WebP, BMP, GIF (first frame of GIFs is used)
  • Drag thumbnails to reorder frames before creating
  • Use the Speed slider to control frame delay (50ms–2s)
  • Your GIF is returned as a direct download — nothing is stored

✎ Draw Mode

  • Draw up to 50 frames on the canvas using marker or eraser
  • Pick a color from the palette or use the custom color picker
  • Three brush sizes: small, medium, large
  • Enable Onion Skin to ghost the previous frame as a drawing guide
  • Click + in the frame strip to add a frame, hover a frame to delete it
  • Click any frame thumbnail to jump to it
  • Hit Clear to wipe the current frame and start over

🔒 Privacy

  • Uploaded images are never written to disk — processed entirely in memory
  • Drawn frames exist only in your browser until you export
  • The server streams your GIF back as raw bytes and immediately discards it
  • Close the tab and everything is gone. Zero traces.
